# VoltStream: Decentralized Smart EV Charging Grid Management System

* Engineered a low-level microgrid simulation engine in Java using robust Object-Oriented Programming (OOP) principles, inheritance, and polymorphism.
* Developed a load-throttling algorithm using proportional scaling logic to optimize hardware power distribution across active charging terminals.
* Implemented a modular business rules engine to calculate dynamic user pricing based on real-time grid load factor metrics.
